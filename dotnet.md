# Dotnet

> [Início](./README.md)

```sh
# Listar pacotes
dotnet list package

# instalar pacotes pelo XML
dotnet restore

# Adicionar pacotes
dotnet add package nome

# Remover pacotes
dotnet remove package nome

# Limpar
dotnet clean

# Executar
dotnet run
dotnet run --urls "https://localhost:5000"
dotnet watch run

# Criar o executável
dotnet build

dotnet --version
dotnet --list-sdks
dotnet --list-runtimes
```
